# Final Year Project

## Project Title
student record management System

## Description
This project is developed using PHP and MySQL. It helps manage student hostel records, room allocation, and fees.

## Features
- Student Registration
- Room Allocation
- Admin Dashboard
- Fee Management

## Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript
- XAMPP Server

## How to Run
1. Copy project to htdocs folder
2. Start Apache & MySQL in XAMPP
3. Import database (.sql file)
4. Open browser and run localhost/project-folder
